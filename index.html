ให้ตั้งค่า media query 3 ขนาดหน้าจอ ในหน้าจอแบ่งเป็น 3 ส่วน 1. มีรูปภาพใหญ่กึ่งกลางจอ 2. ให้แบ่งหน้าจอ เป็น 1, 2 หรือ 3 คอลัมน์ โดยแต่ละคอลัมน์ต้องมีหัวข้อและมีเนื้อหา 3. มีข้อความ ที่เป็นหัวข้อและเนื้อหาอยู่ตรงกลางจอ

หัวข้อและ เนื้อหามีขนาดแตกต่างกันในแต่ละ media query รูปภาพมีความกว้าง 80% ของความกว้างหน้าจอทั้งหมด media query เล็กสุดเป็น 1 คอลัมน์ media query ขนาดกลางและใหญ่มี 2 และ 3 คอลัมน์ตามลำดับ เนื้อหาที่ใส่มีทั้งหมด 6 ช่อง ขนาดตัวหนังสือให้ตกแต่งให้เหมาะสมกับขนาดช่อง
