CSS syntax ประกอบด้วยอะไรบ้าง การเขียน CSS มีกี่แบบอะไรบ้าง selector มีกี่แบบอะไรบ้าง combinator มีกี่แบบอะไรบ้าง

การเขียน CSS มี 3 ลักษณะคือ Inline, Internal และ External โดยสามารถเลือกใช้ได้ความความเหมาะสมเป็นหลัก
1. แบบ Inline คือการเขียนแทรกลงไปในบรรทัดนั้นๆ ของ tag [X]HTML เลย
2. แบบ Internal(embedded) คือเขียนแยก CSS ออกจาก tag [X]HTML แต่อยู่ในหน้าเดียวกันกับไฟล์ [X]HTML โดยแทรก css เอาไว้ที่ header
3. แบบ External คือการเขียน แยก css ออกมาเป็นไฟล์ .css ต่างหาก แล้วนำเข้าไปใช้ใน หน้าเว็บด้วย การเรียกโดย ใช้ โดยการพิมพ์ link rel="stylesheet" href="externalcss.css" type="text/css" ไว้ที่ส่วน header ของหน้าเว็บ

1. Class Selectors เป็น Selector ที่เราต้องกำหนดชื่อของ Selector ขึ้นมาเอง โดย Class Selector การเรียกใช้งานต้องระบุชื่อ Class Selectors ให้กับ Tag ของ HTML จึงจะสามารถใช้งานคุณสมบัติของ Class ที่ได้สร้างไว้ได้
2. ID Selectors เป็น Selector ที่อ้างอิงกับ Attribute ที่ชื่อว่า ID ซึ่งเขียนอยู่ภายใน Tag ของ HTML โดยที่ Tag ของ HTML ใดมี ID ซึ่งตรงกับ ID Selectors ที่กำหนด คุณสมบัติของ CSS ที่กำหนดไว้กับ Selector ก็จะมีผลต่อ Tag นั้น
3. TAG Selectors Selector ที่อ้างอิงกับชื่อ Tag ของ HTML โดยตรง เช่น tag p, tag div, tag h เป็นต้น นั่นหมายความว่า ถ้าหากกำหนดคุณสมบัติให้แก่ TAG Selectors นี้แล้ว Tag ของ HTML ที่มีชื่อเดียวกับ Selectors นี้็จะมีคุณสมบัติของ CSS เหมือนกันหมดทุก Tag ภายในเว็บเพจนั้น

    1. Descendant combinators
    2. Child combinators
    3. Sibling combinators
    3.1 Adjacent sibling combinator
    3.2 General sibling combinator

5. Property for element : display, Height, Width, Position, Float, Align, Flex, Grid

4.1 value ใน display property มีอะไรบ้าง
ตอบ - display: inline; แสดงผลในบรรทัดเดียวกัน เช่น span, a , img, li, ul
- display: block; //แสดงผลคนละบรรทัด เช่น div, h1 — h6, p, form, header, footer, section, br
- display: none; //ซ่อนการแสดงผล คือ ธรรมดา ใช้กับ JavaScript ให้ hide (ซ่อน) และ show (แสดง) องค์ประกอบ (Element) โดยไม่ต้องลบและสร้างใหม่
- visibility:hidden; //ซ่อนการแสดงผล แบบเหลือพื้นที่ไว้
4.2 grid และ flex แตกต่างกันอย่างไร
ตอบ Flex ใช้เพื่อจัดการ layout 1D ส่วน Grid ใช้เพื่อจัดการ layout 2D
Flex เราสามารถจัดวางของต่างๆภายใน layout 1 แถว อาจจะเป็นแนวตั้งหรือแนวนอน จากบนลงล่างหรือจากซ้ายไปขวา
ส่วน Grid เราสามารถจัดของไว้ตรงไหนก็ได้ใน grid layout มีลักษณะเป็น column กับ row
4.3 justify and align แตกต่างกันอย่างไร
ตอบ text-align : จัดให้ชิดซ้าย ขวา หรือวางตรงกลางข้อความ ใช้ค่าเฉพาะคือ left , right, center , justify มีค่า Default คือ left
ส่วน justify คือจัดให้ทั้งซ้ายและขวา ตรงข้ามกันหมด โดยเพิ่มช่องว่างระหว่างคำเพื่อให้ทั้งสองด้านตรงกันหมด

5.1 position คืออะไร และ อะไรบ้าง
ตอบ position คือ property โดยมี value ได้แก่ 1. static เป็นค่าเริ่มต้น 2.relative เป็นการกำหนดตำแหน่งที่อิงความสัมพันธ์กับตำแหน่งเดิม
3.absolute จะกำหนดตำแหน่งโดยอิงกับตำแหน่ง parent ตัวแรก ที่กำหนด position เป็นค่าอื่นที่ไม่ใช่ static
และ 4.fixed ใช้กำหนดตำแหน่งของ element โดยอิงความสัมพันธ์กับตำแหน่งขอบด้านในของบราวเซอร์ และจะไม่เคลื่อนเมื่อมีการเลื่อน scrollbar
5.2 z-index คืออะไรใช้อย่างไร
ตอบ เป็น CSS Property ตัวหนึ่งที่ไว้สำหรับกำหนด ตำแหน่งในแกน Z ของ DOM แต่ละตัว ใช้จัดเรียง elementที่ซ้อนกันและมีผลกับ element ที่กำหนด position ไว้เท่านั้น
ถ้า z-index อันไหนมีค่ามากกว่าก็จะอยู่ด้านบน
